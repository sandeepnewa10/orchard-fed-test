# Webpack App Starter

Frontend environment 

Includes the following features...

- Babel Loader
- Sass Loader
- HTMLWebpackPlugin
- WebpackDevServer
- Asset Resource Loader
- Source Maps
- Caching

## Usage

Run Dev Server (Port 3000)

```
  npm run dev
```

## Build for production

```
npm run build
```

## Demo App

- index.js (main javascript)
- template.html (template)
- sass/style.scss (styles)

## Framework
- Bootstrap framework
